# dev-docs

🦉 一个持续进化的程序员开发文档

**借鉴业界流行的开发规范，整理归纳一份开发文档，帮助养成高效、科学的开发习惯。**

## 出发点

作为开发人员，来到一个团队的第一天，我们最希望看到的是一份详细的开发者指南，它就像指路明灯一样，通过一系列通俗易懂的文字告诉我们应该遵循的相关规范、学习的方向，帮助我们快速融入团队以及系统的成长。

## 目的

不断整理与优化日常开发规范与流程，以提升编程与项目管理能力，同时也期许对尚未明晰开发流程规范的开发者有所帮助。
